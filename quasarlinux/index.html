О QuasarLinux

QuasarLinux — это дистрибутив Linux, разработанный с акцентом на производительность, стабильность и удобство использования. Он сочетает в себе проверенные временем решения с современными технологиями.

OpenRC вместо systemd

Мы используем OpenRC как систему инициализации вместо systemd. Это даёт:

  • Более простая и понятная конфигурация
  • Лучшая совместимость со старыми системами
  • Больший контроль над процессом загрузки
  • Меньше зависимостей и накладных расходов

Кастомность

QuasarLinux создан для тех, кто хочет полного контроля над своей системой:

  • Кастомные ядра (ZEN, LTS, Vanilla)
  • Гибкая настройка инициализации
  • Модульная архитектура
  • Открытый исходный код
quasar@linux:~
$ rc-status
Runlevel: default
agetty                                 [ started ]
dbus                                   [ started ]
network                                [ started ]
sshd                                   [ started ]

# OpenRC - быстрая и простая инициализация
quasar@linux:~$
