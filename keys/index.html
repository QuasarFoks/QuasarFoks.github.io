Публичный ключ для проверки подписи образов

О ключе

Использование: Только для подписи образов ISO

Не используется: В pacman или других пакетных менеджерах

Важно: Этот ключ используется только для подписи официальных образов QuasarLinux и других проектов. Не импортируйте его в pacman.

Как использовать

1

Импортировать ключ

Скопируйте ключ выше и сохраните в файл, затем импортируйте:

gpg --import quasarfoks.pub
2

Проверить образ

Для проверки подписи ISO образа:

gpg --verify image.iso.sig image.iso
3

Проверить подпись

Убедитесь что подпись верна и соответствует отпечатку:

gpg --list-keys --fingerprint

Важное примечание

Этот ключ НЕ используется в pacman!

Ключ предназначен только для проверки подлинности образов ISO. Не добавляйте его в pacman.keyring.

Для пакетов в репозиториях используются отдельные ключи подписи.
