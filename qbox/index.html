Что такое Qbox?

Универсальность

Запускайте пакеты любого дистрибутива в любой системе. DEB на Arch, RPM на Ubuntu и наоборот.

Безопасность

Полная изоляция приложений в контейнерах. Не влияет на основную систему.

Простота

Одна команда для создания контейнера. Автоматическая настройка интеграции.

Qbox решает проблему зависимости от конкретного дистрибутива. Больше не нужно выбирать между любимым дистрибутивом и необходимым программным обеспечением.

Примеры использования

Создать контейнер Ubuntu на Arch Linux

qbox create ubuntu:latest --name my-ubuntu

Установить DEB пакет в контейнере

qbox exec my-ubuntu -- apt install some-deb-package

Запустить GUI приложение из контейнера

qbox run my-ubuntu -- some-gui-application

Список всех контейнеров

qbox list

Qbox автоматически настраивает интеграцию с рабочим столом, буфер обмена и доступ к файлам.
