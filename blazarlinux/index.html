О BlazarLinux

База: LFS

Собирается с нуля на основе Linux From Scratch для максимального контроля и оптимизации.

Минимализм

Только необходимые компоненты. Ничего лишнего для чистоты и производительности.

База для проектов

Будет использоваться как основа для QuasarLinux и других систем сообщества.

BlazarLinux — это фундаментальный проект, создающий минималистичную, оптимизированную базу для будущих дистрибутивов. Собираясь с нуля через LFS, система обеспечивает полный контроль над каждым компонентом.

Планы разработки

Этап 1: Базовая сборка LFS

Создание минимальной рабочей системы на основе Linux From Scratch. Настройка toolchain и базовых компонентов.

Этап 2: Система управления пакетами

Разработка собственного менеджера пакетов или адаптация существующего для LFS-основы.

Этап 3: Интеграция с QuasarLinux

Использование BlazarLinux в качестве базы для следующей версии QuasarLinux.

Этап 4: Поддержка других проектов

Расширение для использования в QuasarOS и других системах сообщества.
